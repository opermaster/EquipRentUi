# Web UI for Equipment Rental Service

Frontend for the [EquipRent API](https://github.com/opermaster/EquipRentApi)

---

## Overview

This project provides a web interface for managing equipment rentals. It allows customers to create rental orders, while managers and administrators can manage orders, equipment, and users.

---

## Catalog Page

The catalog page is used to create rental orders.

Users must:

1. Select rental **start and end dates**
2. Choose available **equipment**
3. Select a **distribution address**
4. Provide **personal information**
